# Personal Academic Website

This repository contains a simple academic website built for GitHub Pages using Markdown and Jekyll.

## How to publish

1. Create a repository named `YOUR-USERNAME.github.io`
2. Upload all files from this folder to the repository
3. In GitHub, go to **Settings > Pages**
4. Set the source to the main branch
5. Save the settings

Your site will then be available at:

`https://YOUR-USERNAME.github.io`

## Files included

- `index.md` — home page
- `research.md` — research overview
- `publications.md` — selected publications and external profiles
- `software.md` — software and data resources
- `contact.md` — professional contact information
- `_config.yml` — GitHub Pages / Jekyll configuration

## What you should edit

Before publishing, replace the placeholder links below wherever appropriate:

- `[ORCID profile URL]`
- `[NASA ADS profile URL]`
- `[Google Scholar profile URL]`
- `[GitHub profile URL]`
- `[Institutional email]`
- `[Link to full CV PDF]`
- publication links marked as placeholders

You can also add:

- a profile image in `assets/img/`
- a PDF CV in `assets/pdf/`

## Notes

This site uses the default **minima** Jekyll theme supported by GitHub Pages, which keeps the repository lightweight and easy to maintain.
